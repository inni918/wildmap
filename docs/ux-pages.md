## 頁面架構圖

```
/                           ← 首頁地圖（MVP-A ✅）
├── /login                  ← 登入/註冊（MVP-A ✅）
├── /forgot-password        ← 忘記密碼（MVP-A ✅）
├── /auth/callback          ← OAuth 回調（MVP-A ✅）
├── /search                 ← 搜尋結果（MVP-B）
├── /profile                ← 個人頁面（MVP-B）
├── /settings               ← 個人設定（MVP-B）
├── /favorites              ← 收藏列表（中期）
├── /notifications          ← 通知中心（中期）
├── /user/[id]              ← 他人頁面（中期）
├── /spot/[id]/report/new   ← 撰寫遊記（中期）
├── /spot/[id]/report/[rid] ← 閱讀遊記（中期）
├── /spot/[id]/booking      ← 訂房系統（遠期）
├── /private-spots          ← 私房地點（遠期）
├── /groups                 ← 群組管理（遠期）
├── /leaderboard            ← 排行榜（遠期）
├── /business
│   ├── /claim/[spotId]     ← 商家聲明（遠期）
│   ├── /dashboard          ← 商家面板（遠期）
│   ├── /plans              ← 訂閱方案（遠期）
│   └── /influencer         ← 達人媒合-商家端（遠期）
├── /influencer
│   └── /dashboard          ← 達人媒合-達人端（遠期）
└── /admin
    ├── /reports            ← 檢舉管理（中期）
    ├── /spots              ← 地標審核（中期）
    ├── /users              ← 用戶管理（中期）
    └── /business           ← 商家審核（遠期）
```
