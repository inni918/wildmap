## 1. 首頁地圖

**路徑**：`/`  
**元件**：`Map.tsx`  
**需登入**：否

### 初始載入
| 行為 | 說明 |
|------|------|
| Loading 狀態 | Header 顯示「載入中...」 |
| 成功 | 地圖載入完成，顯示所有地標 emoji markers，Header 顯示「N 個地點」 |
| 失敗（Supabase 斷線） | 地圖仍可操作，但無地標顯示。Console 記錄錯誤 |
| 空資料 | 地圖顯示但無標記，Header 顯示「0 個地點」 |

### 地圖互動
| 操作 | 行為 |
|------|------|
| 拖曳 | 移動地圖視角 |
| 雙指縮放/滾輪 | 縮放地圖 |
| 點擊地標 emoji | 彈出 Popup 卡片（stopPropagation，不觸發新增） |
| 點擊空白處 | 關閉已開的 Popup → 開啟新增地標 Modal（帶經緯度） |
| 右下角導航控制 | +/- 縮放按鈕 + 羅盤 |

### 視覺
- 游標：`crosshair`（暗示可點擊新增）
- 右下角浮動提示：「點地圖新增地點 ➕」（綠底白字）

## 2. Header 元件

**元件**：`Header.tsx`

### 左側
| 元素 | 互動 |
|------|------|
| 🗺️ Wildmap Logo | 點擊刷新首頁 |

### 右側
| 元素 | 條件 | 互動 |
|------|------|------|
| 「載入中...」 | loading=true | 靜態文字 |
| 「N 個地點」 | loading=false | 顯示篩選後的地標數量 |
| 灰色圓形脈動 | authLoading=true | 佔位動畫 |
| 用戶頭像 | 已登入 | 點擊展開下拉選單 |
| 「登入」按鈕 | 未登入 | 點擊導向 `/login` |

### 用戶下拉選單
| 元素 | 互動 |
|------|------|
| 暱稱 | 顯示 display_name 或 email |
| Lv.X · N 分 | 顯示等級和積分 |
| 登出 | 清除 session，關閉選單 |

**點擊外部**：自動關閉選單（mousedown 監聽）

## 4. 特性篩選面板

**元件**：`FeatureFilter.tsx`

### 收合狀態
| 元素 | 互動 |
|------|------|
| ▸ 特性篩選 | 點擊展開面板 |
| 數字徽章（綠色） | 顯示已選特性數量（0 時不顯示） |

### 展開狀態
| 元素 | 互動 |
|------|------|
| ▾ 特性篩選 | 點擊收合 |
| 清除全部（紅色文字） | 清除所有選取，僅在有選取時顯示 |
| 六大類別標題 | 彩色圓點 + 類別名稱 |
| 特性 Tag 按鈕 | 點擊 toggle 選取/取消 |

**特性 Tag 樣式**：
- 未選取：灰底灰框（`bg-gray-50 border-gray-200`）
- 已選取：綠底綠框（`bg-green-100 border-green-400`）

**篩選邏輯**：
- 多選取交集：地標必須同時滿足所有選中特性
- 特性需 ≥60% 投票通過才算「有此特性」
- Loading：無特殊指示（前端計算）

**空結果**：地圖上無標記，Header 顯示「0 個地點」

## 5. 地標 Popup

**最大寬度**：260px

### 內容
| 元素 | 說明 |
|------|------|
| emoji + 名稱 | 類型 emoji + truncate 名稱 |

| 免費/付費 | 綠色（免費）/ 橘色（付費），null 時不顯示 |
| 描述 | 有則顯示 |
| 地址 | 📍 前綴，有則顯示 |
| 經緯度 | 🧭 格式 xx.xxxx, xx.xxxx |
| 「查看特性詳情 →」 | 點擊開啟 SpotDetail 面板 |

**關閉**：點擊 X 或點擊地圖其他位置

## 6. 地標詳情面板

**元件**：`SpotDetail.tsx`  
**觸發**：從 Popup 點擊「查看特性詳情」  
**動畫**：底部滑入（`animate-slide-up`），關閉時滑出（`translate-y-full`，200ms）  
**最大高度**：80vh

### Header 區
| 元素 | 說明 |
|------|------|
| 拖曳把手 | 灰色橫條（視覺提示可關閉） |
| emoji + 名稱 | truncate |
| 類型 + 品質標籤 | 同 Popup |
| 描述 | 最多 3 行（`line-clamp-3`） |
| 經緯度 | 📍 格式 |
| 特性 icon 摘要 | 緊湊排列已確認特性 icon |

### Tab 列
| Tab | 說明 |
|-----|------|
| 特性一覽 | 唯讀展示已確認/待確認特性 |
| 投票特性 | 登入用戶可投票（未登入顯示 🔒） |

### 特性一覽 Tab
| 狀態 | 顯示 |
|------|------|
| Loading | spinner + 「載入特性資料...」 |
| 有特性 | 按六大類別分組，色塊 Tag + ✅/❓ |
| 無特性 | 🤷 + 「尚無特性資料」+ 引導切換到投票 Tab |

**特性 Tag 樣式**：
- 已確認（≥60%）：類別顏色底 + `✅`
- 待確認（<3 票）：灰底 + `❓` + 半透明
- hover title 顯示確認人數

### 投票特性 Tab
| 狀態 | 顯示 |
|------|------|
| 未登入 | 🔒 + 「登入後即可投票」 |
| 已登入 | 所有適用特性列表，每項 ✅有 / ❌沒有 按鈕 |
| 已投過 | 顯示自己的投票 + 當前結果 |

**關閉方式**：
- 點擊 × 按鈕
- 點擊背景遮罩

## 7. 新增地標 Modal

**元件**：`AddSpotModal.tsx`  
**觸發**：點擊地圖空白處

### 未登入狀態
| 元素 | 互動 |
|------|------|
| 🔒 需要登入 | 提示文字 |
| 「使用 Google 登入」 | 觸發 Google OAuth |
| 「先不要」 | 關閉 Modal |

### 已登入狀態 — 表單

| 欄位 | 類型 | 必填 | 驗證規則 | 錯誤訊息 |
|------|------|------|---------|---------|
| 地點類型 | 6 宮格按鈕 | ✅ | 預設 camping | — |
| 地點名稱 | text input | ✅ | 非空白 | 「請輸入地點名稱」 |
| 簡介 | textarea（3 行） | ❌ | — | — |
| 地址 | text input | ❌ | — | — |
| 聯絡電話 | tel input | ❌ | — | — |
| 社群連結 | url input | ❌ | — | — |

**類型選擇樣式**：
- 選中：綠框綠底（`border-green-500 bg-green-50`）
- 未選：灰框

### 送出
| 狀態 | 按鈕文字 | 行為 |
|------|---------|------|
| 正常 | ✅ 新增地點 | 提交表單 |
| Loading | 新增中... | disabled + 半透明 |
| 成功 | — | 關閉 Modal + 刷新地圖 |
| 失敗 | — | 紅字「新增失敗，請再試一次」 |

**提示**：底部灰字「💡 地點建立後，可以在詳情頁為各項特性投票」

**關閉方式**：
- 點擊 × 按鈕
- 點擊背景遮罩（`e.target === e.currentTarget`）

## 8. 登入/註冊頁

**路徑**：`/login`  
**元件**：`login/page.tsx`

### 頁面結構
- Logo 區：🗺️ Wildmap + 副標題「探索台灣最棒的戶外秘境」
- Tab 切換：登入 / 註冊
- 表單區
- 分隔線：「或使用其他方式」
- 社群登入按鈕
- 底部連結：「← 返回地圖」

### 登入 Tab

| 欄位 | 類型 | 驗證 | 錯誤訊息 |
|------|------|------|---------|
| Email | email input | required, email 格式 | 瀏覽器原生驗證 |
| 密碼 | password input | required | — |

| 元素 | 互動 |
|------|------|
| 忘記密碼？ | 導向 `/forgot-password` |
| 登入按鈕 | 提交 → loading 態「登入中...」→ 成功導向 `/` / 失敗顯示紅色錯誤區 |

### 註冊 Tab

| 欄位 | 類型 | 驗證 | 錯誤訊息 |
|------|------|------|---------|
| 顯示名稱 | text input | 非空白 | 「請輸入顯示名稱」 |
| Email | email input | required | — |
| 密碼 | password input | ≥ 6 字元 | 「密碼至少需要 6 個字元」 |
| 確認密碼 | password input | 與密碼一致 | 「密碼不一致」 |

| 元素 | 互動 |
|------|------|
| 建立帳號按鈕 | 提交 → loading 態「註冊中...」→ 成功導向 `/` / 失敗顯示錯誤 |

### 社群登入
| 按鈕 | 互動 |
|------|------|
| 使用 Google 帳號繼續 | 觸發 `signInWithGoogle()`，跳轉 Google OAuth |
| 使用手機號碼登入 | 觸發 toast「手機登入功能即將推出」（3 秒自動消失） |

### 錯誤區
- 紅底紅框圓角區塊（`bg-red-50 border-red-200`）
- 切換 Tab 時清除錯誤
- 錯誤文字紅色（`text-red-600`）

### Toast
- 固定頂部居中
- 深灰底白字
- 3 秒自動消失
- 淡入動畫（`animate-fade-in`）

## 全域互動規範

### Toast 提示
| 類型 | 樣式 | 持續 |
|------|------|------|
| 成功 | 綠底白字 | 3 秒 |
| 錯誤 | 紅底白字 | 5 秒 |
| 資訊 | 深灰底白字 | 3 秒 |
| 警告 | 橘底白字 | 5 秒 |

**位置**：螢幕頂部居中（目前實作：深灰底）

### Loading 狀態
| 元件 | Loading 表現 |
|------|------------|
| 地圖 | Header 顯示「載入中...」 |
| 特性 | spinner + 「載入特性資料...」 |
| 按鈕 | disabled + 文字變「...中」（如「登入中...」）|
| 列表 | 骨架屏（`animate-pulse`） |
| 頭像 | 灰色圓形脈動 |

### 錯誤狀態
| 類型 | 處理 |
|------|------|
| 表單驗證 | 紅底紅框錯誤區塊，位於表單上方 |
| API 失敗 | toast 或內嵌錯誤訊息 |
| 網路斷線 | 目前無特殊處理（未來加 offline banner） |
| 權限不足 | 導向登入頁或顯示鎖定提示 |

### 空資料狀態
| 場景 | 顯示 |
|------|------|
| 地圖無地標 | 顯示空地圖 + 「0 個地點」 |
| 特性無資料 | 🤷 + 引導投票 |
